# finance-dashboard
A simple and interactive Finance Dashboard to track your income and expenses. This project provides a clean and responsive UI to manage personal finances easily.

## 🚀 Features

- ➕ Add new transactions (Income & Expense)
- 📊 View total balance, income, and expenses
- 📈 Balance trend visualization (Line Chart)
- 🥧 Income vs Expense breakdown (Pie Chart)
- 📄 Transaction history table
- 🎨 Responsive and user-friendly UI

---

## 🛠️ Tech Stack

- React.js
- JavaScript (ES6)
- Tailwind CSS / CSS
- Recharts

## ⚙️ Installation & Setup

### 1. Clone the repository
git clone https://github.com/your-username/finance-dashboard.git


### 2. Navigate to the project folder
cd finance-dashboard

### 3. Install dependencies
npm install

### 4. Run the project
npm start
